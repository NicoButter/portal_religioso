# Caminando por la Fe

<p align="center">
  <img src="public/images/logo.png" alt="Logo Caminando por la Fe" width="200" />
</p>

**Caminando por la Fe** es un portal web moderno dedicado a la espiritualidad, la comunidad y la música litúrgica. Construido con las últimas tecnologías web, ofrece un espacio interactivo para la reflexión, la oración y el servicio pastoral.

## 🌟 Características Principales

- **📖 Reflexiones Diarias**: Artículos y meditaciones sobre espiritualidad, vida diaria y fe, organizados por categorías.
- **🎼 Transpositor de Acordes Web**: Herramienta avanzada para músicos y coros que permite transponer canciones en tiempo real y visualizar diagramas de acordes.
- **🙏 Rezos y Devociones**: Una colección de oraciones estructuradas para el acompañamiento espiritual.
- **⛪ Galería de Celebraciones**: Espacio visual para compartir los momentos significativos de la comunidad (Bautismos, Bodas, Misas, etc.).
- **📅 Lecturas y Novedades**: Acceso a las lecturas del día y noticias relevantes de la comunidad.
- **📱 Diseño Responsivo**: Experiencia optimizada para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: [Next.js 15+](https://nextjs.org) (App Router), [React 19](https://reactjs.org)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: [Tailwind CSS 4](https://tailwindcss.com)
- **Iconografía**: [Lucide React](https://lucide.dev)
- **Despliegue**: [Vercel](https://vercel.com)

## 🚀 Instalación y Desarrollo

Sigue estos pasos para ejecutar el proyecto localmente:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/portal_religioso.git
   cd portal_religioso
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Ver el sitio:**
   Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 📁 Estructura del Proyecto

```text
├── app/                  # Rutas, páginas y lógica del App Router
│   ├── components/       # Componentes de UI reutilizables
```
